Vue VS. Angular VS. React

  • Vue

    • 轻量级UI库 渐进式框架
    • 多种使用方式，适应性强
    • 高效变更检测，虚拟DOM，默认性能优化
    • 单文件组件，灵活的文件组织方式
    • 强大易用的cli工具
    • 大型项目实践相对较少
    • 对ES Next、TypeScript支持较别扭

  • Angular

    • Google背书，有Angular.js的成功经验
    • 面向未来，原生TS支持
    • 完善全家桶，开箱即用，适合偏后端系统
    • 复杂度高（纯业务开发简单），基本强制TS
    • 体积相对大
    • 易用性相对差一些，高大上但少些亲和性

  • React

    • 小巧，灵活，思维的创新
    • 专注UI层，协力构建生态
    • 拥抱ES Next，工具链齐全
    • JSX是把双刃剑
    • React Native

没有最好的，只有最合适的！强扭的瓜不甜！
